# HTML标签

##常用标签
````
<head> </head> 头部
<body> </body> 身体

<title> </title> 网页名称
<h1> </h1>一级 1标题
<h1 align="center"> 标题居中 </h1>
<a href="地址"> </a> 超链接
<p> </p> 段落 
<img src = "地址" alt="规定图像的代替文本" width="宽度" height="高度">
<br> 换行
<hr> 水平线
<b> </b> 加粗文本
<i> </i> 斜体
<code> </code> 代码块
<sub> </sub> 下标
<sup> </sup> 上标
<strong> </strong> 和 <em> </em> 意味着你要呈现的文本是重要的
<pre> </pre> 自定义文本块
<del> </del> 删除符
<ins> </ins> 下划线
 ````
