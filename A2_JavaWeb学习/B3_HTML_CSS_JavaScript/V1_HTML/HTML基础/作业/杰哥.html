周杰伦        

1979年1月18日台湾省新北市中国

  周杰伦（Jay Chou），1979年1月18日出生于台湾省新北市，祖籍福建省泉州市永春县，中国台湾流行乐男歌手、音乐人、演员、导演、编剧，毕业于淡江中学。

  2000年发行首张个人专辑《Jay》。2001年发行的专辑《范特西》奠定其融合中西方音乐的风格 [1] 。

  2002年举行“The One”世界巡回演唱会 [2] 。2003年成为美国《时代周刊》封面人物 [3-4] 。2004年获得世界音乐大奖中国区最畅销艺人奖 [333] 。

  2005年凭借动作片《头文字D》获得金马奖、金像奖最佳新人奖 [6] 。2006年起连续三年获得世界音乐大奖中国区最畅销艺人奖 [7] 。

  2007年自编自导的文艺片《不能说的秘密》获得金马奖年度台湾杰出电影奖 [8] 。
